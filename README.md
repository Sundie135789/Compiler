Vertex is a small, custom programming language designed to be experimental. This repository contains the interpreter for Vertex, written in C++.

Features: 
Executes Vertex source files (files with extension .vx)
Supports basic arithmetic, integer and string variable support, and conditionals
Minimal and fast -> Fast for learning language design
Descriptive error messages for logical and syntactical errors

Usage
For now, only REPL mode is supported.
Syntax for using REPL mode is given below.
> ./vertex
